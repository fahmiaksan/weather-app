## Project

This project was built using react js and tailwind css<br><br>
## How to Run the Project
```

# Clone the repo
git clone https://github.com/fahmiaksan/wheather-app.git

# Enter the project directory
cd wheather-app

# Install dependencies
npm install

# Run the project
npm start<br>

```
